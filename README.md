Este repositório contém o código-fonte da camada de cliente (front-end) do projeto da A3 da Unidade Curricular de Sistemas Distribuídos e Mobile. Esta aplicação é uma refatoração do sistema de Gerenciamento de Empréstimo de Ferramentas, utilizando Java Swing para a interface gráfica. A principal alteração é a remoção do acesso direto ao banco de dados; este cliente agora consome a API RESTful exposta pelo back-end para todas as operações de cadastro, consulta e relatórios.
